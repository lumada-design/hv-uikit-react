# useHvFilters

The `useHvFilters` and `useHvGlobalFilter` allows filtering rows for a given column or all columns respectively.

Check our React Table's [`useFilter`](https://react-table-v7-docs.netlify.app/docs/api/useFilter) and [`useGlobalFilter`](https://react-table-v7-docs.netlify.app/docs/api/useGlobalFilter)
documentation for configuration details.

Another example using these hooks can be found in the [table section page](./?path=/docs/visualizations-table-table-section--docs) under the
Table Filter section.
