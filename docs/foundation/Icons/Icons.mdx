# Icons

The UI Kit provides a library of SVG icons packaged as React components for use in applications.

The icons can be imported from the `@hitachivantara/uikit-react-icons` package:

```tsx
import { CheckboxCheck } from "@hitachivantara/uikit-react-icons";
```

## Icon Sprites

The UI Kit icons package supports the usage of SVG icon sprites, by providing:

- An `HvIconSprite` component that allows loading icons from an SVG sprite sheet.
- SVG icon sprites under the `dist/sprites` folder.

_Note: Using a sprite sheet for the icons is useful when there's a need to pre-load all icons from the library. If this isn't the case, using the individual icons is preferred._

```tsx
import { HvIconSprite } from "@hitachivantara/uikit-react-icons";

const MyComponent = () => (
  <HvIconSprite
    iconName="CheckboxCheck" // icon id from the sprite
    spriteUrl="/public/icons.svg" // URL to the sprite sheet
  />
);
```
