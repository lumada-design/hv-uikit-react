# Theming

The UI Kit components provides the two Design System themes: **Dawn** and **Wicked**.

## Custom themes

The UI Kit theme object is an enhanced Material-UI like theme object,
the basic structure of a Material-UI theme can be found in [here](https://material-ui.com/customization/default-theme/).
UI Kit appends to the material theme an `hv` property which contains extra definitions required for the components,
these definitions can be found in [here](https://github.com/lumada-design/hv-uikit-react/blob/v2.x/packages/themes/src/dawn.json).

If you want to override one of the UI Kit themes you must build your custom theme using the `themeBuilder` utility function.

The function receives a JSON object with the same structure as the definitions found in [here](https://github.com/lumada-design/hv-uikit-react/blob/v2.x/packages/themes/src/dawn.json). **Currently, it must be a complete theme definition,** it will not patch/fill in missing values.

The `themeBuilder` will output a Material-UI theme object with the values and utility functions properly set so the theme then can be passed to the `HvProvider`.

```
import { themeBuilder } from "@hitachivantara/uikit-react-core";

const myRawCustomTheme = {
  "name": "custom",
  "palette": {
    "accent": {
      ...
      "acce2": "#33CE33",
      "acce2h": "#4AB573",
      ...
    },
    "atmosphere": {
      "atmo1": "#E3F1F6",
      ...
      "atmo3": "#BAE3F1",
      ....
    },
    "semantic": {
      "sema1": "#4AB573",
      "sema2": "#33ABCC",
      "sema3": "#F5996E",
      ...
      "sema15": "#9672C1",
      ...
    },
    ...
  }
}

const myCustomTheme = themeBuilder(myRawCustomTheme);

<HvProvider theme={myCustomTheme}>
  <MyApp>
<HvProvider/>
```
