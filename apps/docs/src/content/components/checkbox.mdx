### Color

You can use the [`color`](/docs/styling#color-prop) prop to style the color of the checkbox.

```tsx live
<div className="flex gap-sm">
  <HvCheckBox defaultChecked color="primary" />
  <HvCheckBox defaultChecked color="positive" />
  <HvCheckBox defaultChecked color="coral" />
  <HvCheckBox defaultChecked color="#c73aa8" />
</div>
```

### Controlled

You can share state control from a parent component. Clicking **Checkbox 1** has no effect, while clicking **Checkbox 2** updates the state and toggles both checkboxes.

```tsx live
import { useState } from "react";

export default function Demo() {
  const [isChecked, setIsChecked] = useState(false);

  return (
    <>
      <HvCheckBox checked={isChecked} label="Checkbox 1" />
      <HvCheckBox
        checked={isChecked}
        onChange={(_, checked) => setIsChecked(checked)}
        label="Checkbox 2"
      />
    </>
  );
}
```
