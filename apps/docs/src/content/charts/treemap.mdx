### Custom ECharts options

You can customize the chart using ECharts options.  
In this example, the item styles of the tree map are customized to control appearance and visual distinction.

  name="Treemap"
  onOptionChange={(option) => {
    option.series[0].visibleMin = 100;
    option.series[0].upperLabel = {
      show: false,
      height: 20,
    };
    option.series[0].levels = [
      {
        itemStyle: {
          borderColor: "#777",
          borderWidth: 0,
          gapWidth: 1,
        },
        upperLabel: {
          show: false,
        },
      },
      {
        itemStyle: {
          borderColor: "#555",
          borderWidth: 5,
          gapWidth: 1,
