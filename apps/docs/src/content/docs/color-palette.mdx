# Color Palette

The **UI Kit color palette** is designed to provide consistent visual language for **data-rich** enterprise applications.
Each color spans from **50** (lightest) to **950** (darkest), offering flexibility and accessibility across **light and dark modes**.

<Callout type="warning">
  Avoid using raw palette values directly in your code. Use **color tokens**
  instead (e.g. `theme.colors.primary`) to ensure visual consistency, theme
  support, and accessibility across the system.
</Callout>

If needed (e.g. for data visualizations or custom elements), you can access palette values via the `theme.palette` object:

```tsx
import { theme } from "@hitachivantara/uikit-react-core";

const MyComponent = () => (
  <div style={{ color: theme.palette.blue[500] }}>
    This text uses a raw palette color.
  </div>
);
```
