# Color Tokens

Color tokens can be accessed through the `theme.colors` object:

```tsx
import { theme } from "@hitachivantara/uikit-react-core";

const MyComponent = () => (
  <div style={{ color: theme.colors.primary }}>
    This text uses the semantic primary color.
  </div>
);
```
