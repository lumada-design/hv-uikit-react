### With scroll

If you give fixed dimensions to the panel and the content exceeds the dimensions, the panel will automatically add a scrollbar.

```tsx live
export default function Demo() {
  const chars = "abcdefghijklmnopqrstuvwxyz";
  return (
    <HvPanel
      className="size-300px whitespace-nowrap tracking-1rem"
      tabIndex={0}
    >
      {[...chars].map((char, i) => (
        <div key={i}>{i === 0 ? chars : char}</div>
      ))}
    </HvPanel>
  );
}
```
