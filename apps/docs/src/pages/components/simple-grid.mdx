### Breakpoints

Use the `breakpoints` prop to define the number of columns and spacing for different widths.

```tsx live
<div className="w-full">
  <HvSimpleGrid
    cols={2}
    breakpoints={[
      { minWidth: 1600, cols: 4, spacing: "md" },
      { minWidth: 1300, cols: 3, spacing: "md" },
      { minWidth: 1100, cols: 2, spacing: "sm" },
      { minWidth: 800, cols: 1, spacing: "sm" },
    ]}
  >
    {[...Array(6).keys()].map((i) => (
      <HvCard bgcolor="atmo2">
        <HvCardContent className="flex items-center justify-center">
          Item {i + 1}
        </HvCardContent>
      </HvCard>
    ))}
  </HvSimpleGrid>
</div>
```
