# UI Kit Components

UI Kit Components are the building blocks of your application.
They are designed to be reusable and customizable, allowing you to create a consistent look and feel across your app.

Most components are available in the [core package](https://npm.im/@hitachivantara/uikit-react-core), available in the public npm repository:

```sh
npm i @hitachivantara/uikit-react-core
```

The components can then be imported and used as shown below:

```tsx
import { HvButton, HvTypography } from "@hitachivantara/uikit-react-core";
```

```tsx live
<div className="flex gap-sm">
  <HvButton variant="primary">Button</HvButton>
  <HvTypography variant="title3">Typography</HvTypography>
</div>
```

Besides the _main_ core package, we also provide the following packages for more specific use-cases:

- [`@hitachivantara/uikit-react-icons`](https://npm.im/@hitachivantara/uikit-react-icons) ([Docs](/docs/icon-library)): A collection of React icons for the NEXT Design System.
- [`@hitachivantara/uikit-react-lab`](https://npm.im/@hitachivantara/uikit-react-lab) ([Docs](/components/lab)): A collection of experimental React components.
- [`@hitachivantara/uikit-react-viz`](https://npm.im/@hitachivantara/uikit-react-viz) ([Docs](/charts)): A collection of chart components for use in your application.
- [`@hitachivantara/uikit-react-code-editor`](https://npm.im/@hitachivantara/uikit-react-code-editor) ([Docs](/components/code-editor)): A specialized wrapper around the [Monaco Editor](https://microsoft.github.io/monaco-editor).
