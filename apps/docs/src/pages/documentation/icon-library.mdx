# NEXT Icon Library

UI Kit also provides a library of SVG icons packaged as React components for use in applications,
meant for the [NEXT Design System users](https://designsystem.hitachivantara.com/6c705d900/p/31e481-icons).

The icons are available in the `@hitachivantara/uikit-react-icons` package, and follow the [`HvIconContainer`](/components/icon-container) API.

```tsx
import { Alert } from "@hitachivantara/uikit-react-icons";
```

<br />

<Playground
  Component={Alert}
  componentName="Alert"
  controls={{
    size: { defaultValue: "M" },
    color: { defaultValue: "warning" },
  }}
/>
